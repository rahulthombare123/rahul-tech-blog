Creating Dark Mode with CSS Variables

CSS February 2, 2026 Rahul

Dark mode is a popular feature. Using CSS custom properties (variables) makes it easy to implement and maintain.

Step 1: Define CSS Variables

Create a :root block for light mode, and a separate class for dark mode.

:root {
    --bg-color: #ffffff;
    --text-color: #333333;
    --primary: #0066cc;
}

.dark-mode {
    --bg-color: #222222;
    --text-color: #f0f0f0;
    --primary: #66b0ff;
}

Step 2: Use Variables Throughout Your CSS

body {
    background-color: var(--bg-color);
    color: var(--text-color);
}

Step 3: Toggle with JavaScript

const toggleBtn = document.getElementById('theme-toggle');
toggleBtn.addEventListener('click', () => {
    document.body.classList.toggle('dark-mode');
    // Save preference in localStorage
});

Tip: Save the user's preference in localStorage so dark mode persists across page reloads.

Step 4: Handle System Preference (optional)

Use the prefers-color-scheme media query to respect the user's OS setting.

if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
    // enable dark mode
}

Conclusion

CSS variables make theme switching a breeze. This approach is scalable and easy to maintain. Give it a try on your own site!
