10 JavaScript Array Methods You Must Know

JavaScript March 10, 2026 Rahul

Arrays are everywhere in JavaScript. These ten methods will make your code cleaner, more readable, and more functional.

1. map()

Creates a new array by applying a function to each element.

const doubled = [1,2,3].map(x => x * 2); // [2,4,6]

2. filter()

Returns a new array with elements that pass a test.

const evens = [1,2,3,4].filter(x => x % 2 === 0); // [2,4]

3. reduce()

Accumulates values into a single result.

const sum = [1,2,3].reduce((acc, x) => acc + x, 0); // 6

Tip: reduce() can also group or transform data â€“ very powerful!

4. forEach()

Executes a function for each element (no return).

[1,2,3].forEach(x => console.log(x));

5. find()

Returns the first element that satisfies a condition.

const firstEven = [1,2,3].find(x => x % 2 === 0); // 2

6. some() / every()

Test if any/all elements pass a condition.

[1,2,3].some(x => x > 2); // true
[1,2,3].every(x => x > 0); // true

7. includes()

Checks if an array contains a value.

[1,2,3].includes(2); // true

8. push() / pop()

Add/remove elements from the end.

const arr = [1]; arr.push(2); // [1,2]; arr.pop(); // [1]

9. shift() / unshift()

Add/remove from the beginning.

arr.unshift(0); // [0,1]; arr.shift(); // [1]

10. slice() / splice()

slice() returns a portion (no mutation). splice() changes the array (add/remove).

[1,2,3,4].slice(1,3); // [2,3]
arr.splice(1, 1); // removes one element at index 1

Conclusion

Practice these methods daily. They will transform how you work with data.
